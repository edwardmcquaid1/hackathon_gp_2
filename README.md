#  Workload Fire Map

A project to help apprentices better organise their time, help the programme team understand workload bottlnecks
and build heatmap timelines from:
- Delivery plans
- Assessment Dates
- Real Apprenticeship Logging

This is group 2's submission for the 07/07/2026 block release Hackathon.
